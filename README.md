# married.af

Wedding website for Amanda Zhang and Francis Williams, hosted on GitHub Pages.

## Development

This site is built with Jekyll, a static site generator.

### Setup

```bash
bundle install
```

### Build

```bash
bundle exec jekyll build
```

### Serve locally

```bash
bundle exec jekyll serve
```

Visit http://localhost:4000 to view the site.

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the main branch. The custom domain `married.af` is configured via the CNAME file.
